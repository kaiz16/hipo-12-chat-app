Chat app
### API address
`https://chat-app-hipo-12.herokuapp.com`

### Install nodemon (-g means global)
```
npm install nodemon -g
```

## Run the backend
```
cd backend
npm install
nodemon index.js
```
It should run on `localhost:3000`

## Run the frontend
```
cd frontend
npm install 
npm run serve
```
Should run on `localhost:8080`
